# 🎮 Sistema de Gerenciamento de Jogos

Um sistema simples em PHP para gerenciar uma coleção de jogos via linha de comando. Permite cadastrar, listar, buscar, editar e remover jogos, além de exibir estatísticas.

## 📋 Funcionalidades

- **Cadastrar** novos jogos com validação de dados
- **Listar** todos os jogos em ordem alfabética
- **Buscar** jogos por nome (busca parcial)
- **Editar** informações de jogos existentes
- **Remover** jogos da coleção
- **Exibir estatísticas** (total, jogos mais novos/antigos, status de atualização)

## 🚀 Como usar

### Pré-requisitos
- PHP 7.0 ou superior
- Terminal/Console

### Executando o programa

1. Salve o código em um arquivo (ex: `jogos.php`)
2. Abra o terminal na pasta do arquivo
3. Execute:
```bash
php jogos.php
